3D Scatter Plot

The **3D Scatter Plot** project is an interactive web application that visualizes randomly generated points in a three-dimensional space. The project demonstrates the concepts of 3D graphics, including point generation, rotation, and projection. Users can interact with the scatter plot by changing the viewing angles and selecting different projection methods.

Project Objective

The main objectives of this project are:

* Understand how 3D points are represented in a coordinate system.
* Visualize randomly generated data in 3D space.
* Apply Yaw, Pitch, and Roll rotations to view the point cloud from different angles.
* Compare Orthographic and Perspective projection techniques.
* Create an interactive and animated 3D visualization using HTML5 Canvas.

 Project Workflow

* Generate random 3D points using Python.
* Store the generated points in a **data.json** file.
* Load the JSON data into JavaScript.
* Apply Yaw, Pitch, and Roll rotations.
* Convert 3D coordinates into 2D using projection techniques.
* Display the points on an HTML5 Canvas.
* Continuously update the visualization using animation.

Key Features

* Random generation of 1000 3D points.
* Interactive sliders for Yaw, Pitch, and Roll.
* Auto Rotate option for continuous rotation.
* Orthographic Projection mode.
* Perspective Projection mode.
* Smooth visualization using HTML5 Canvas.

Technologies Used

* HTML
* CSS
* JavaScript
* Python
* JSON
* HTML5 Canvas

Applications

* Computer Graphics
* Data Visualization
* Educational Demonstrations
* Scientific Data Representation
* 3D Visualization

Conclusion

The 3D Scatter Plot project provides a simple and interactive way to understand the fundamentals of 3D graphics.
It combines random point generation, 3D transformations, and projection techniques to create an engaging visualization that helps users learn how three-dimensional data is displayed on a two-dimensional screen.
