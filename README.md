Developed a 3D Scatter Plot using HTML, CSS, JavaScript, Python, and JSON.
Generated random 3D points using Python and stored them in a data.json file.
Implemented Yaw, Pitch, and Roll rotations to view the point cloud from different angles.
Added Orthographic and Perspective projection methods for different visualization styles.
Created an interactive interface with rotation sliders and an Auto Rotate feature to visualize the 3D scatter plot dynamically.
